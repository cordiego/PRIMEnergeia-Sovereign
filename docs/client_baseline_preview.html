2How We Calculate Recoverable Capital

For each settlement interval, we compute the difference between what your plant could have injected (HJB-optimal) and what it actually injected, valued at the real-time locational marginal price:

// Recovery formula per interval
Δ Capitali = (Optimal_MWi − Actual_MWi) × LMPi × Δt

// HJB value function (mathematically provable optimal)
Vt + minu { L(x, u) + ∇V · f(x, u) } = 0

// Subject to Swing Equation constraint
df/dt = (Pm − Pe − D·Δf) / 2H
