Operator Override

You always have control. The software never overrides your safety systems.

From SCADA HMI (always takes priority):

  1. Set Operator Override tag → TRUE
  2. Software stops writing setpoints instantly
  3. Dashboard shows: ⚠️ OPERATOR OVERRIDE ACTIVE
  4. Your manual controls take full priority
  5. Set back to FALSE to restore

From Dashboard:

  1. Grid Control page → SHADOW MODE toggle
  2. Software switches to read-only
  3. No setpoints written until re-enabled
